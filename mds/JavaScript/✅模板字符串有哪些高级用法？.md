# ✅模板字符串有哪些高级用法？

# 典型回答

模板字符串（Template Literals）是ES6引入的字符串语法，使用反引号（\`）替代单引号或双引号。相比传统字符串，模板字符串提供了以下核心能力：

**基本特性：**

1. **嵌入式表达式**：`${expression}`语法嵌入任意JavaScript表达式
2. **多行字符串**：直接支持换行，无需`\n`拼接
3. **标签模板（Tagged Templates）**：通过自定义函数处理模板字符串

**高级用法：**
- 嵌套模板
- 条件表达式嵌入
- 函数调用嵌入
- 模板字符串作为函数参数
- 标签模板实现字符串转义、国际化、SQL构建等

# 扩展知识

## 嵌入式表达式详解

```javascript
const name = 'Alice';
const age = 25;

// 嵌入变量
console.log(`Name: ${name}`);

// 嵌入运算
console.log(`${name} is ${age > 18 ? 'adult' : 'minor'}`);

// 嵌入函数调用
const getUser = () => ({ name: 'Bob', role: 'admin' });
console.log(`User: ${getUser().name}, Role: ${getUser().role}`);

// 嵌入方法链
const items = [1, 2, 3];
console.log(`Sum: ${items.reduce((a, b) => a + b, 0)}`);

// 嵌套模板
const nested = `Outer ${`Inner ${name}`}`;
console.log(nested); // Outer Inner Alice
```

## 标签模板（Tagged Templates）

标签模板是模板字符串最强大的特性，允许用函数处理模板字符串：

```javascript
// 基本标签函数
function highlight(strings, ...values) {
  return strings.reduce((result, str, i) => {
    const value = values[i] ? `<strong>${values[i]}</strong>` : '';
    return result + str + value;
  }, '');
}

const user = 'Alice';
const role = 'admin';
const result = highlight`User ${user} has role ${role}`;
// "User <strong>Alice</strong> has role <strong>admin</strong>"
```

## 标签模板的实际应用

```javascript
// 1. SQL防注入
function escapeSQL(strings, ...values) {
  return strings.reduce((result, str, i) => {
    const sanitized = values[i] ? values[i].replace(/'/g, "''") : '';
    return result + str + sanitized;
  }, '');
}
const name = "O'Brien";
const query = escapeSQL`SELECT * FROM users WHERE name = '${name}'`;
// SELECT * FROM users WHERE name = 'O''Brien'

// 2. HTML转义
function htmlEscape(strings, ...values) {
  const escape = (str) => str
    .replace(/&/g, '&amp;')
    .replace(/</g, '&lt;')
    .replace(/>/g, '&gt;')
    .replace(/"/g, '&quot;');
  return strings.reduce((result, str, i) => {
    return result + str + (values[i] ? escape(values[i]) : '');
  }, '');
}

const userInput = '<script>alert("xss")</script>';
const safe = htmlEscape`<div>${userInput}</div>`;
console.log(safe); // <div>&lt;script&gt;alert("xss")&lt;/script&gt;</div>

// 3. 国际化（i18n）
function i18n(strings, ...values) {
  // 模拟语言包
  const translations = {
    'Hello': '你好',
    'Welcome': '欢迎'
  };
  return strings.reduce((result, str, i) => {
    const translated = translations[str.trim()] || str;
    return result + translated + (values[i] || '');
  }, '');
}
console.log(i18n`Hello, ${name}!`); // 你好, Alice!

// 4. 带样式的控制台输出
function styled(strings, ...values) {
  return strings.reduce((result, str, i) => {
    const style = values[i] ? `%c${values[i]}` : '';
    return result + str + style;
  }, '');
}
console.log(styled`Error: ${'red'} occurred`);
```

## 原始字符串（Raw Strings）

通过`String.raw`标签函数可以获取原始字符串（不处理转义字符）：

```javascript
// 普通模板字符串
console.log(`Hello\nWorld`);
// Hello
// World

// 原始字符串
console.log(String.raw`Hello\nWorld`);
// Hello\nWorld

// 实际应用：正则表达式
const regex = String.raw`\d+\.\d+`;
console.log(regex); // \d+\.\d+

// 文件路径
const path = String.raw`C:\Users\Alice\Documents`;
console.log(path); // C:\Users\Alice\Documents
```

## 自定义标签函数中获取原始字符串

```javascript
function logRaw(strings) {
  console.log('解析后的:', strings[0]);
  console.log('原始的:', strings.raw[0]);
}

logRaw`Hello\nWorld`;
// 解析后的: Hello
//            World
// 原始的: Hello\nWorld
```

## 模板字符串与性能

```javascript
// 模板字符串在运行时求值
const fn = (name) => `Hello, ${name}!`;
// 等价于：
const fn2 = (name) => 'Hello, ' + name + '!';

// 在V8引擎中，模板字符串的性能与字符串拼接相当或更优

// 避免在循环中创建复杂的模板
// 不推荐：
for (let i = 0; i < 1000; i++) {
  const message = `Item #${i}: ${getDetail(i)}`;
}

// 推荐：使用数组join
for (let i = 0; i < 1000; i++) {
  const message = ['Item #', i, ': ', getDetail(i)].join('');
}
```

## 模板字符串的限制

```javascript
// 不能直接引用自身
// const self = `${self}`; // ReferenceError

// 不能包含未转义的回引号
// const bad = `can't use \` directly`; // 需要转义：\`

// 三元运算符需要括号
const a = 1;
console.log(`value: ${a > 0 ? 'positive' : 'negative'}`); // 正确
// console.log(`value: ${a > 0 ? 'positive' : 'negative'}`); // 同样的效果

// 模板字符串中的表达式不能是语句
// console.log(`${if (true) { 'yes' }}`); // SyntaxError
```

## 最佳实践

- 使用模板字符串替代字符串拼接
- 复杂表达式前先计算，保持模板字符串可读性
- 利用标签模板处理需要对字符串做额外处理的场景
- 使用`String.raw`处理包含大量反斜杠的字符串
