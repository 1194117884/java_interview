# ✅如何实现暗黑模式？

# 典型回答

暗黑模式（Dark Mode）实现的核心是**在不同主题下切换 CSS 颜色值**。现代前端实现暗黑模式主要有以下几种方案：

## 方案对比

| 方案 | 原理 | 优点 | 缺点 | 适用场景 |
|------|------|------|------|---------|
| **CSS 变量 + 媒体查询** | 使用 `prefers-color-scheme` 检测系统主题 | 纯 CSS，无 JS 依赖 | 无法手动切换 | 跟随系统主题 |
| **CSS 变量 + JS 切换** | 通过 JS 切换根元素的 class/data 属性 | 支持手动切换 | 需要少量 JS | 需要用户控制主题 |
| **Tailwind dark: 类** | 使用框架的 dark: 前缀 | 开发效率高 | 依赖框架 | Tailwind 项目 |
| **CSS 自定义属性覆盖** | 在不同选择器下重新定义变量值 | 灵活，易维护 | 变量管理需规范 | 大型项目 |

## 推荐方案：CSS 变量 + data 属性切换

```css
/* 基础样式 */
:root {
  --color-bg: #ffffff;
  --color-text: #1a1a1a;
  --color-primary: #3498db;
  --color-border: #e0e0e0;
  --color-card: #f5f5f5;
  --shadow: 0 2px 8px rgba(0, 0, 0, 0.1);
}

/* 暗黑模式 */
[data-theme="dark"] {
  --color-bg: #1a1a2e;
  --color-text: #e0e0e0;
  --color-primary: #64b5f6;
  --color-border: #333;
  --color-card: #16213e;
  --shadow: 0 2px 8px rgba(0, 0, 0, 0.3);
}

/* 使用变量 */
body {
  background: var(--color-bg);
  color: var(--color-text);
}

.card {
  background: var(--color-card);
  border: 1px solid var(--color-border);
  box-shadow: var(--shadow);
}
```

```javascript
// JS 控制切换
const themeToggle = document.querySelector('.theme-toggle');

// 切换主题
themeToggle.addEventListener('click', () => {
  const currentTheme = document.documentElement.getAttribute('data-theme');
  const newTheme = currentTheme === 'dark' ? 'light' : 'dark';
  document.documentElement.setAttribute('data-theme', newTheme);
  localStorage.setItem('theme', newTheme);
});

// 初始化：优先使用用户上次的选择
const savedTheme = localStorage.getItem('theme');
if (savedTheme) {
  document.documentElement.setAttribute('data-theme', savedTheme);
} else if (window.matchMedia('(prefers-color-scheme: dark)').matches) {
  document.documentElement.setAttribute('data-theme', 'dark');
}
```

## 媒体查询方案（跟随系统）

```css
/* 自动跟随系统设置 */
:root {
  --bg: white;
  --text: black;
}

@media (prefers-color-scheme: dark) {
  :root {
    --bg: #121212;
    --text: #eee;
  }
}

body {
  background: var(--bg);
  color: var(--text);
}
```

## 图片适配

暗黑模式下，亮色图片可能过亮，需要降低亮度：

```css
/* 图片暗黑适配 */
[data-theme="dark"] img {
  filter: brightness(0.8) contrast(1.2);
}

/* 对特定图片不应用滤镜 */
[data-theme="dark"] img.no-dark {
  filter: none;
}
```

## 防止主题闪烁（FOIT）

将主题恢复脚本放在 `<head>` 中同步执行：

```html
<head>
  <script>
    // 在页面渲染前执行，避免闪烁
    (function() {
      const theme = localStorage.getItem('theme');
      if (theme === 'dark' ||
        (!theme && window.matchMedia('(prefers-color-scheme: dark)').matches)) {
        document.documentElement.setAttribute('data-theme', 'dark');
      }
    })();
  </script>
</head>
```

## 过渡动画

添加主题切换时的平滑过渡：

## 在框架中使用

### Vue

```vue
<template>
  <div :data-theme="theme">
    <button @click="toggleTheme">切换主题</button>
  </div>
</template>

<script>
export default {
  data() {
    return {
      theme: localStorage.getItem('theme') || 'light',
    };
  },
  watch: {
    theme(val) {
      localStorage.setItem('theme', val);
    },
  },
  methods: {
    toggleTheme() {
      this.theme = this.theme === 'dark' ? 'light' : 'dark';
    },
  },
  mounted() {
    const prefersDark = window.matchMedia('(prefers-color-scheme: dark)');
    if (!localStorage.getItem('theme') && prefersDark.matches) {
      this.theme = 'dark';
    }
  },
};
</script>
```

### React

```jsx
function ThemeProvider({ children }) {
  const [theme, setTheme] = useState(() => {
    const saved = localStorage.getItem('theme');
    if (saved) return saved;
    return window.matchMedia('(prefers-color-scheme: dark)').matches
      ? 'dark' : 'light';
  });

  useEffect(() => {
    document.documentElement.setAttribute('data-theme', theme);
    localStorage.setItem('theme', theme);
  }, [theme]);

  const toggleTheme = () => {
    setTheme(prev => prev === 'dark' ? 'light' : 'dark');
  };

  return (
    <ThemeContext.Provider value={{ theme, toggleTheme }}>
      {children}
    </ThemeContext.Provider>
  );
}
```

## 设计系统集成

在组件库层面统一管理颜色令牌：

```scss
// 颜色令牌映射
$color-tokens: (
  'bg-primary': (
    light: #ffffff,
    dark: #121212,
  ),
  'text-primary': (
    light: #1a1a1a,
    dark: #e0e0e0,
  ),
  'brand': (
    light: #3498db,
    dark: #64b5f6,
  ),
);

// 生成 CSS 变量
:root {
  @each $token, $values in $color-tokens {
    --color-#{$token}: #{map-get($values, 'light')};
  }
}

[data-theme="dark"] {
  @each $token, $values in $color-tokens {
    --color-#{$token}: #{map-get($values, 'dark')};
  }
}
```

## 无障碍注意事项

1. **对比度要求**：暗黑模式下文字和背景的对比度仍需满足 WCAG AA 标准（4.5:1）
2. **尊重用户偏好**：默认应跟随系统设置
3. **提供切换选项**：让用户有权选择
4. **不要只依赖媒体查询**：给用户手动切换的能力
